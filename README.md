# Mestrado

Este repositório destina-se ao compartilhamento dos códigos desenvolvidos e dados coletados durante o período de realização do Mestrado em Automação de Sistemas de Energia do Programa de Pós-Graduação em Engenharia Elétrica pela Universidade Estadual Paulista "Júlio de Mesquita Filho" - Faculdade de Engenharia de Bauru.
O Projeto conta com apoio financeiro do Conselho Nacional de Desenvolvimento Científico e Tecnológico (CNPq).
